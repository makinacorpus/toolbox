Saisie de géométries avec Leaflet

Leaflet permet de saisir des géométries sur une carte interactive

Description

Fonctionnalités

  • création et modification d'objets géométriques sur une carte : points, lignes, polygones
  • utilisable sur ordinateur, tablette ou smartphone
  • transfert en JSON des informations vers l'application, le serveur ou la base de données

Démo

Particularités de la démo

  • Utilisation au sein du CMS Plone (mais peut être intégré dans toute page web)
  • Les géométries dessinées sont stockées dans la base du CMS
